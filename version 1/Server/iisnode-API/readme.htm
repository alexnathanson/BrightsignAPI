Device Check-in

node.js apps that use the popular express framework can be hosted in IIS. Installation instructions at https://github.com/Azure/iisnode.

Some specific settings and permissions in IIS may need to be changed as well. When I installed it originally, the home directory handler mappings in IIS needed to be set to read/write<
- Feature Delagation -> Handler Mappings = Read/Write

Copy the entire deviceInfo directory to the Program Files/iisnode/www directory (or whatever IIS is redirecting to)

view logs at iisnode
debug the application at deviceInfo.js/debug (requires WebKit enabled browser)
